# Photopea Automation using Playwright

This project automates the creation of a design in Photopea using Playwright.

## Features
- Canvas creation (1280x960, transparent)
- Text automation
- Shape creation
- Image upload & background removal
- Drag and drop example
- File export (.png, .jpg, .psd)
- Preview vs download validation

## Tech Stack
- Playwright
- Node.js

## Setup

1. Clone repository
2. Install dependencies

npm install

3. Run automation

npx playwright test
